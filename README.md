Dashboard for the QUIQQER System

Package name:

    quiqqer/dashboard


Features
--------

- Quick overview of important statistics like:
    - Project-, Page-, User-, Group-Count
    - Last changed pages
    - Last logged in users
    - System information
    - Filesystem information
    - CRON history
    - and many more...
- Can be extended by other packages
